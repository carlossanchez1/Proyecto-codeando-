HTML Imagenes:

Imágenes HTML se definen con la etiqueta < img> .
La marca < img> no necesita cierre y su sintaxis correcta es:

< img src="imag.jpg">

donce src corresponde al inglés Search y es el recorrido del que el navegador saca la imagen (en este caso "imag.jpg").
Como hemos señalado, esta marca es única en el sentido de que NO lleva como cierre el correspondiente < /img>.
